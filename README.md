# 📋 To-Do List

O sistema consiste em uma aplicação de gerenciamento de tarefas integrada a uma API REST, permitindo que usuários realizem cadastro, login e gerenciamento completo de suas tarefas.

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js

## Recursos Utilizados

- API REST
- JWT Authentication
- CORS
- Fetch API

## ✨ Funcionalidades

### 👤 Autenticação

* Cadastro de usuários
* Login de usuários
* Armazenamento de token JWT
* Rotas protegidas por autenticação

### 📝 Gerenciamento de Tarefas

* Criar tarefas
* Visualizar tarefas
* Editar tarefas
* Excluir tarefas

---

## ⚙️ Executando o Backend

1. Abra o VS Code.
2. Navegue até a pasta:

```text
teaching/todo-list/server
```

3. Clique com o botão direito na pasta **server**.
4. Selecione **Open in Integrated Terminal** (Abrir no Terminal Integrado).
5. Execute o comando:

```bash
npm install
```

6. Após a instalação das dependências, execute:

```bash
npm run dev
```

7. O servidor será iniciado em:

```text
http://localhost:8000
```

---

## 🌐 Executando o Frontend

1. Abra a pasta:

```text
teaching/todo-list/client
```

2. Abra o arquivo `index.html` (ou a página inicial do projeto).
3. Clique com o botão direito no arquivo.
4. Selecione **Open with Live Server**.

Também é possível executar o frontend utilizando `npx serve` ou por meio do recurso de visualização do VS Code(🌐).


## 🌐 Comunicação com a API

A comunicação entre cliente e servidor é realizada utilizando a Fetch API.
